# 1. Tabella delle derivate delle funzioni elementari
Vedere [[Esempi di derivate]] per eventuali *"dimostrazioni"* di alcune derivate.

|:------------------:|:-------------------------:|
| $$c \in \mathbb{R}$$ |            $$0$$            |
|   $$\text{id}_x$$    |            $$1$$            |
|       $$x^n$$        |        $$nx^{n-1}$$         |
|       $$e^x$$        |           $$e^x$$           |
|      $$\ln x$$       |       $$\frac{1}{x}$$       |
|      $$\sin x$$      |         $$\cos x$$          |

|    $$\arctan x$$     |     $$\frac{1}{1+x^2}$$     |
